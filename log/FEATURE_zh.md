## 功能

VoceSpace 提供全面的视频会议体验，拥有先进的 AI 功能和丰富的自定义选项。

**基本控制**
- [x] 麦克风预览和测试
- [x] 摄像头预览及设备选择
- [x] 实时视频预览
- [x] 自定义用户名输入及自动生成
- [x] 一键加入房间

**高级设置**
- [x] 音量调节和测试
- [x] 多麦克风设备选择
- [x] 视频模糊强度控制及实时预览
